# Another Todo App

A CLI tool for user to create and manage their Todo lists

Working on:

- Todo API
  - item
  - List
    - Complete
    - Add/Create
    - Edit/Modify
    - Delete
    - Save
    - Get
- CLI implementation

- REST API Server that lets users modify Todo list with HTTP methods
  - View Todos
  - Add Todo
  - Edit/Modify Todos

- Use SQLite for data persistence
